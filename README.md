## Methylation Pipeline 

To map reads and call methylation, we used [BsBolt](https://github.com/NuttyLogic/BSBolt). Duplicate reads were removed using [UMI-tools](https://github.com/CGATOxford/UMI-tools). 

#### dependencies 
- BsBolt
- Samtools 
- UMI-tools
